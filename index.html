How to Deploy This Page to GitHub Pages

  1. Create a free account on GitHub.
  2. After logging in, click on New repository to create a new project repository.
  3. Give your repository a name (e.g., voidara-landing), set it to Public, and click Create repository.
  4. On the repository page, click Add file > Upload files, then upload the index.html file containing this landing page code.
  5. After uploading, commit the changes.
  6. Go to Settings > Pages in the repository sidebar.
  7. Under Source, choose main branch (or your default branch) and save.
  8. Wait a few minutes. Your page will be live at https://your-github-username.github.io/your-repository-name/.

How to Customize the Landing Page

  • Change colors: Edit the background property in the CSS body selector to adjust the gradient colors.
  • Update links: Replace the URLs in the href attributes of the two buttons in the <a> tags.
  • Swap YouTube videos: Update the src attribute of the iframe with any new YouTube embed link.
  • Modify contact info: Change the phone number inside the <footer> tag.

Feel free to personalize the page further to align it with your branding and message. You can add your logo, change fonts, or add more content as needed.
